# BetterDummy Virtual Display

**BetterDummy Virtual Display** is an open-source macOS application that creates virtual dummy displays for advanced screen management. As a powerful **BetterDummy Screen Resolver**, it unlocks missing scaled resolutions on external monitors, including 4K and ultrawide displays. This **BetterDummy Mac Dummy** tool allows you to create headless displays that persist even after reboot or logout. With **BetterDummy Headless Adapter**, you can use Sidecar with any Mac, enable screen streaming without physical monitors, and control display arrangements precisely.

## Features

- **Virtual Displays**: The **BetterDummy Virtual Display** creates any resolution and refresh rate you need.
- **Resolution Unlock**: Use **BetterDummy Screen Resolver** to access HiDPI scaling on any external monitor.
- **Headless Operation**: As a **BetterDummy Mac Dummy**, run your Mac without any physical screen attached.
- **Sidecar Enabler**: This **BetterDummy Headless Adapter** makes Sidecar work on older or non-retina Macs.
